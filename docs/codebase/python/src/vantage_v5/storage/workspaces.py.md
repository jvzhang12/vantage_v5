# `src/vantage_v5/storage/workspaces.py`

This module provides the legacy workspace persistence layer in the codebase. `WorkspaceStore` maps each workspace to a single Markdown file named `{workspace_id}.md` under the configured workspaces directory, and `WorkspaceDocument` is the in-memory record that callers receive back after load or save. In the product canon, this is the whiteboard store behind the older `workspace` implementation name.

The store reads the full file content as UTF-8, derives a title from the first `# ` heading when one exists, and otherwise falls back to a title-cased version of the workspace ID. Saving always creates parent folders, writes the Markdown back to disk, and then reloads the document so the returned record reflects the persisted file state. If an existing workspace is already a Scenario Lab branch and the new content omitted the `## Scenario Metadata` block, the store re-injects the existing branch metadata before writing so edits do not accidentally strip the durable branch identity.

The module now also exposes a small Markdown-first scenario metadata contract for branch workspaces. `inject_scenario_metadata()` inserts a stable `## Scenario Metadata` block near the top of a document, `parse_scenario_metadata_block()` parses that generic block back out, and `parse_workspace_scenario_metadata()` normalizes the branch-specific fields that Scenario Lab writes today. Loaded `WorkspaceDocument`s carry that parsed data in `scenario_metadata` when the workspace is a scenario branch.

This keeps workspace storage filename-first and Markdown-first. There is still no frontmatter or separate workspace database, but Scenario Lab branch workspaces can now advertise durable fields like `scenario_kind`, `base_workspace_id`, `comparison_question`, `branch_label`, `comparison_artifact_id`, `scenario_namespace_id`, and `namespace_mode` in a form other layers can reopen later. As the whiteboard canon continues to replace workspace language, this module remains the compatibility layer for the live draft surface.
