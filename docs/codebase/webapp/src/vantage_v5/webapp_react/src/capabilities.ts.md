# `src/vantage_v5/webapp_react/src/capabilities.ts`

Frontend helpers for rendering app capability and artifact-action information.

## Purpose

- Translate backend capability/action identifiers into compact product-facing labels.
- Keep proposed update cards and receipt copy artifact-neutral across calendar, tasks, and future app surfaces.
- Avoid duplicating capability wording across React components.

## Coverage

- Labels for artifact actions such as calendar edits and task updates.
- Human-readable operation names and capability status summaries.
- Small formatting helpers used by the main Vantage UI and inspection surfaces.
